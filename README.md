# Windows 11 Lock Screen Simulator

The windows 11 lock screen simulator can be used to simulate the windows 11 lock screen. It is designed to be a visual replica of the lock screen, and includes animations to make it feel real. Uses include testing the lock screen or screenshoting the lock screen, and is very simple to use.

## Use

To use the simulator, there are various things that you can do.

1. **Add Users:** Along the left side of the screen, you should see a *Users* panel. By clicking the **+** icon, you can create another default user. To remove a user, press the red trash icon next to their name.
2. **Configure Users:** To configure a user, press their box on the left panel. Then, near the bottom of the screen you should see some settings for the user. These are the things you can do with them:
    - **Name** To change the name of a user, you can change the text inside the *Name* box. To save the name, you must press the `Enter` key on your keyboard.
    - **Verification Method** To change the verification method of a user, you can select from either `PIN` or `Password`. If you select `PIN`, you have an extra option to select the length of the `PIN` using the *PIN Digits* slider.
    - **Internet Mode** To change the internet mode of a user, you can use the dropdown to chose a mode. To see what mode your computer uses, look near the right end of the taskbar.
    - **Background/Icon** To change the background or user icon for a user, you can upload an image by pressing *Upload Image*.
    - **Saving** Whenever you change the configurations for a user, the data will be stored and can be retrieved when you reload or reopen the page.
3. **Running:** To run the simulation, press the fullscreen icon in the main box in the middle of the screen. Then, you can use it like the default windows lock screen. To exit, press the `Esc` key on your keyboard.
4. **Output:** When you enter either a password or a PIN into the simulation, the text that you enter will be printed to the right side of the screen.

## Installation

To install the simulation, simply follow the following two steps:

1. Download the `.html` file from the GitHub repository.
2. Run the `.html` file
